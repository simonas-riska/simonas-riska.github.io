# Incident Response Home Lab

## About
This site documents a realistic Incident Response home lab focused on SOC and DFIR analyst workflows.

The emphasis is on **investigation, decision-making, documentation, and detection improvement**, not on offensive exploitation.

**Status:** In progress (actively maintained)  
**Last updated:** 2026-02-08

## Focus Areas
- SIEM alert triage and prioritization
- Windows endpoint investigations
- Timeline reconstruction
- Containment and remediation decisions
- Post-incident reporting
- Detection improvement aligned with MITRE ATT&CK

## What this demonstrates
- Alert → investigation → response → reporting workflow
- Windows telemetry reasoning (Sysmon + Event Logs)
- MITRE ATT&CK-aligned analysis
- Detection improvement after incident
